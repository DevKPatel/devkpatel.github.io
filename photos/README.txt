PHOTOS GO HERE 💖
==================

1. Put your pictures in THIS folder.
2. Name them exactly:  photo1.jpg   photo2.jpg   photo3.jpg
   (.jpg or .png both work — if you use .png, update the names in script.js too)
3. To add MORE photos, open script.js and add lines to the "photos" list, e.g.:
       { file: "photo4.jpg", caption: "our trip 🌊" },

Tips:
- Square-ish photos look best (they get cropped to a square).
- Keep each photo under ~1 MB so the page loads fast on her phone.
- If a photo is missing, the page just shows a cute "add photo" placeholder
  instead of breaking — so nothing ever looks broken.
